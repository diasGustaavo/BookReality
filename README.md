# BookReality

## Description 

An AR app that reads the book cover and places a significant item from the story on it. Currently supporting [Inferno](https://en.wikipedia.org/wiki/Inferno_(Brown_novel)) and [The Times Complete History of the World](https://harpercollins.co.uk/products/the-times-complete-history-of-the-world-richard-overy?variant=40156094922830), but more books can easily be added. Made for iOS and iPadOS >= 13.
